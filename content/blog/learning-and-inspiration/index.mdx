我的個人網站終於完成了！在這過程中讓我摸到了很多之前沒碰過的技術，我想在這裡分享一些使用的技術，以及一些給我靈感的網站。

## 使用技術
這裡的都是一些我之前未曾用過的技術，這也讓我在開發過程中非常快樂，有種讓自己升等的感覺：
- [next-intl](https://next-intl-docs.vercel.app)：實現網站的多國語系支援。
- [Framer motion](https://www.framer.com/motion/)：非常方便的動畫庫，雖然 tailwindcss 做出動畫也不難，但一些比較複雜的動畫效果使用 framer-motion 會更方便和簡潔。
- [Neon](https://neon.tech/)：一個 postgresql platform，在我的網站用來實現**文章的瀏覽次數**。
- [Prisma](https://www.prisma.io/)：這個 orm(Object Relational Mapping) library 主要是在操作 mysql 時能輕鬆與資料庫進行交互，而且提供了很多直觀的 api，大大化簡了資料庫操作的流程。
- [Velite](https://velite.js.org/)：用來管理我的部落格文章和專案的內文，且能夠很簡單的定義好一個文章需要的資料，例如文章標題、描述、發布日期等等。

## 靈感來源

我在開發個人網站時深刻得體會到，UI 設計是一件很耗時間的事情。當你感到缺乏靈感時，以下這些網站可以給你很多選擇，節省許多設計的時間：
- [Awwwards](https://www.awwwards.com/)
- [Dark Mode Design](https://www.darkmodedesign.com)
- [Godly](https://godly.website/)
- [UI Verse](https://uiverse.io/)
- [Aceternity UI](https://ui.aceternity.com/components)

此外，以下是一些功能豐富且 UI 設計美觀的網站：
- [leerob.io](https://leerob.io)
- [delba.dev](https://delba.dev/)
- [jahir.dev](https://jahir.dev/)
- [honghong.me](https://honghong.me/)
