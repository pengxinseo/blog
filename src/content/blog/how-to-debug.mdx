## 代码调试

- 调试 sever 代码

  - root 目录

    ```bash
    cd packages/server/bin
    ```

  - 打开 [Javascript Debug Terminal](https://code.visualstudio.com/docs/nodejs/nodejs-debugging#_javascript-debug-terminal)

  - 启动

    ```bash
    node --inspect ./run start
    ```

  - 打开 [http://localhost:3000/](http://localhost:3000/)
